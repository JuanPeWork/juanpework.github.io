Array

Distintas formas de crear un array

Existen dos formas de crear un array:

var vector = new Array();

var vector = [ ];

En los dos casos estamos creando un array sin longitud. Podemos establecer la longitud de nuestro array pero hay que tener en cuenta que estas posiciones estarían vacias:

var vector = new Array(5);

Es más recomendable la creación del array sin el new.

Distintos tipos de arrays

Hay dos tipos de arrays:

  • Array paralelos: los arrays paralelos utilizan el mismo índice para referirse a términos similares, es decir, pueden recorrerse en el mismo bucle.
  • Array multidimensionales: Más conocidos como matrices, consiste en un array que contiene otros array. La dimesion de cada uno representan las filas y las columnas. A la hora de recorrer el array para mostrar los valores de cada uno de los arrays que contiene se debe hacer con el uso de dos bucles.

Distintas formas de eliminar elementos de un array

Hay dos formas de eliminar datos de un array:

  • splice(): este operador es el más recomendado. Con el podemos indicar la posición a partir de la cual comenzará a eliminar, donde dejara de eliminar y los elementos que deseamos añadir después de eliminar los elementos.
  • delete():Este operador nos elimina el contenido de la posición especificada, pero el inconveniente es que nos deja la posición vacía.
